Vocabulary

  • Tag: Also known as opening and closing tags, most often represented by angle brackets with associated tag names contained inside. Closing tags begin with a forward slash

  • Body Element: Defines the body of the document, which contains the content that is visible to the user. Body Elements are also examples of Semantic HTML elements.

  • Child Elements: Are elements nested inside other elements.

  • Semantic HTML Elements: Name of the element describes its content and the role it plays on the webpage. Makes YOUR HTML easier to read by other developers and the browser. Semantic HTML elements also play an important role in making sure your webpage is accessible to the widest variaty of users.

  • Decleration: Tells the browser what document type it should expect when it is opened. Declerations are not case sensitive as HTML elements are.

  • Metadata: Data about our webpage that the browser uses to effectively serve our webpage to users. Contained in the header not typically seen by users, but contains important information about accessibility, search engines, and performance.

  • SEO: Search Engine Optimization

Elements

HTML elements work together to build the structure of a webpage and provide a framework that makes it easy for a user to navigate a webpage.


:head
Contains information, or metadata, that helps the browser render the page correctly. Elements contained within the head are not visible when a user looks at a webpage.
:body
Contains the content that is shown to the user. All the elements that we want the user to see and interact with must be placed inside this element.
:title
Is required and is extremely important for SEO.
:base
Defines the base URL for a webpage.
:link
Connects an external resource to the HTML document.
:meta
Defines metadata such as the character set, description, keywords, author, and viewport.
:script
Embeds client-side scripts in the HTML document.
:div
Creates a division in the page.
:span
Creates an inline division.
:p
Paragraph or block of text.
:ul
Unordered list or bullet points.
:ol
Ordered list
:li
List items.
:img
Contains information about images displayed on a webpage
:h1-6
Represents level of heading in a given text block. 1 being largest text size and 6 being the smallest.
:br
Creates a line of empty space
:a
Stands for anchor, creates links to the same page or other webpages.

Attributes

href
Points to the url for the link
src
This is the most important image attribute because it defines the source or location of the image file
alt
contains the text string that describes the appearance and functionality of the image

HTML attributes are always defined in the beginning element, never the closing element
